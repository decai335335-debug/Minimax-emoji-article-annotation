# 申论材料标注技能

## 一、技能目标

对国考、省考等申论材料进行系统化标注，识别并标注核心观点、问题分析、对策措施、背景介绍、评分要点等不同类型信息，便于后续归纳总结、提炼答案要点。

## 二、标注格式规范

### 基本格式

| 格式 | 语法 | 示例 |
|:---|:---|:---|
| **加粗关键词** | `**text**` | `**电商法**` `**营业执照**` |
| **括号标注** | 反引号 + `【emoji 文字】` | `【⭐核心论点】` |
| **连续标注** | 依次写出，不加分隔 | `【🔧措施】` `【🚀成效】` |
| **普通括号** | 无反引号，保持原样 | `(如"7×24")` |

### 标注位置

**核心原则：标注要紧跟在被标注内容后面，不换行**

正确：`电商平台的经营者须取得营业执照。`【⭐核心变化：电商法将小代购纳入监管范围】

错误：
```
电商平台的经营者须取得营业执照。
【⭐核心变化】（换行了，错误）
```

### Emoji使用

Emoji必须放在括号内，格式：`【emoji 文字】`

## 三、十大标注类型与Emoji

### 1. 核心观点类 ⭐💎🏆✨
用于文章开篇定调、结尾升华、直接对应采分点的核心句

| 标注内容 | 用途 |
|:---|:---|
| 【⭐核心论点】 | 全文主旨，直接对应2分采分点 |
| 【💎核心句】 | 指出关键对比关系（如"有形"vs"无形"） |
| 【🏆核心经验】 | 经验总结，可用于对策类题目 |
| 【✨核心转变】 | 思想、观念的深刻变化 |

### 2. 问题分析类 ⚠️❗🔴💢
用于标注负面现状、困境、挑战、典型案例

| 标注内容 | 用途 |
|:---|:---|
| 【⚠️核心问题】 | 问题本质/根本原因 |
| 【⚠️问题表现】 | 问题的具体体现 |
| 【⚠️典型案例】 | 印证问题的具体事例 |
| 【⚠️挑战】 | 实施过程中的困难 |
| 【🔴类型X】 | 问题分类编号 |

### 3. 对策措施类 🔧✅🛠️📋
用于具体做法、实施方案、制度创新

| 标注内容 | 用途 |
|:---|:---|
| 【🔧措施/对策】 | 具体做法 |
| 【🔧创新做法】 | 创新性措施 |
| 【🔧运营模式】 | 运作机制 |
| 【🔧制度保障】 | 制度性安排 |
| 【✅经验X】 | 可推广的经验 |

### 4. 成效反馈类 🚀📈💫🎉
用于积极变化、直接效果、群众感受

| 标注内容 | 用途 |
|:---|:---|
| 【🚀直接成效】 | 具体可量化的成果 |
| 【📈数据变化】 | 数字统计类成效 |
| 【💫间接成效】 | 衍生效果 |
| 【🎉群众反馈】 | 群众评价原文 |

### 5. 背景介绍类 🌐📍🏛️
用于说明性信息，明确标注非采分点

| 标注内容 | 用途 |
|:---|:---|
| 【🌐背景】 | 宏观背景、政策背景 |
| 【📍地点/人物】 | 具体场景介绍 |
| 【🏛️政策依据】 | 政策文件来源 |

### 6. 数据统计类 💯📊🔢
用于数字、比例、统计类信息

| 标注内容 | 用途 |
|:---|:---|
| 【💯数据】 | 关键数字统计 |
| 【📊比例】 | 百分比、占比 |
| 【🔢规模】 | 数量级描述 |

### 7. 分析推理类 🔍💡🧩
用于分析过程、洞察、原因推导

| 标注内容 | 用途 |
|:---|:---|
| 【🔍分析】 | 调研分析过程 |
| 【💡洞察/启示】 | 深层思考 |
| 【⚠️原因X】 | 问题原因分析 |

### 8. 概念定义类 📖🔑
用于术语解释、本质揭示

| 标注内容 | 用途 |
|:---|:---|
| 【📖概念】 | 专业术语定义 |
| 【🔑本质】 | 事物核心本质 |
| 【💡内涵】 | 深层含义 |

### 9. 定位定性类 🎯📌
用于明确角色定位、工作定位

| 标注内容 | 用途 |
|:---|:---|
| 【🎯定位】 | 工作/项目定位 |
| 【📌原则】 | 指导原则 |
| 【🌟价值】 | 意义/价值阐述 |

### 10. 评分指导类 📝⚠️
用于标注采分点、答题提示、易错提醒

| 标注内容 | 用途 |
|:---|:---|
| 【📝采分点】 | 潜在答案要点 |
| 【📝答题提示】 | 答题方向指导 |
| 【⚠️注意】 | 易错点提醒 |
| 【⚠️区分】 | 需要注意的区分点 |

## 五、标注操作流程

### 五步标注法

```
┌─────────────────────────────────────┐
│  1. 通读段落                        │
│     理解段落核心含义和逻辑关系       │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  2. 识别信息类型                    │
│     对照十大分类确定所属类别         │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  3. 提取关键词并加粗                │
│     原词术语、核心名词/动词短语       │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  4. 添加括号标注                     │
│     【emoji 标注内容】紧跟句子后     │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  5. 合并连续标注                     │
│     同句多类型时依次写出不加分隔     │
└─────────────────────────────────────┘
```

### 判断逻辑树

```
材料内容
    │
    ├─ 核心观点/主旨句？ → 【⭐核心论点】
    │
    ├─ 问题/困境/挑战？ → 【⚠️核心问题/表现】
    │
    ├─ 措施/做法/经验？ → 【🔧措施/✅经验】
    │
    ├─ 成效/变化/反馈？ → 【🚀成效/🎉群众反馈】
    │
    ├─ 数据/统计/比例？ → 【💯数据】
    │
    ├─ 背景/说明/介绍？ → 【🌐背景】+ 明确"非采分点"
    │
    ├─ 概念/定义/本质？ → 【📖概念/🔑本质】
    │
    ├─ 定位/原则/价值？ → 【🎯定位/📌原则】
    │
    └─ 答题提示/易错点？ → 【📝提示/⚠️注意】
```

## 六、标注示例

### 示例1：核心观点标注

> 《电子商务法》的立法过程可能是40年来中国法律界最为艰难的一次博弈。

`【🌐背景：电商法立法漫长博弈过程】`

> 电商法将营业执照明文规定在法律中并不难理解，这也是部分平台目前的做法。

`【⭐核心变化：电商法将营业执照法定化】`

### 示例2：问题分析标注

> 过去，企业和个人来办事，遇到不确定的问题都要到办事窗口进行咨询，有时候只是为了确认一个小问题，往往要在办事窗口排很长的队，浪费了办事人的时间，也占用了中心受理业务的资源。

`【⚠️问题表现：办事咨询需排队，浪费时间、占用资源】`

### 示例3：对策措施标注

> 新区行政服务大厅迅速响应并设立了专门的"解疑专员"，为办事市民答疑解难。

`【🔧措施1：设立"解疑专员"提供答疑服务】`

> 新区已经与市政府公众信息网管理中心和市不动产登记局进行了沟通，在新区行政服务大厅设置查询打印终端，开放相关房产登记信息的查询打印服务，避免企业重复跑腿。

`【🔧措施2：数据共享——设置查询终端、开放房产信息】`

### 示例4：成效反馈标注

> "真没想到，别的地方最少也得3年，我们只用一年半的时间就实现了回迁！"

`【🎉群众反馈：回迁速度超预期】`

> 2019年，玉皇农业科技有限公司已形成集特色种养、农产品加工贮运、休闲农业和乡村旅游于一体的综合性产业，年销售收入突破1000万元。

`【🚀直接成效：年销售收入突破1000万元】`

### 示例5：连续标注（多类型）

> 玉皇庙村流转全村所有土地，成立了玉皇农业科技有限公司，但由于缺少懂现代农业的专业人才，公司连年亏损。

`【🔧创业模式：土地流转成立公司】【⚠️问题：缺乏专业人才】`

### 示例6：背景非采分点标注

> 近年来，D县有两名村党组织书记被提拔进入乡镇领导班子。

`【🌐背景：干部激励政策，非本题直接采分点】`

### 示例7：数据标注

> 每天读书2小时以上的学生仅占8.7%，1小时以下的高达70.4%。

`【💯数据：阅读2小时以上8.7%、1小时以下70.4%】`

### 示例8：分类编号标注

> 有一类干部多是部门副职或是老前辈，年龄大了，临近退休。

`【🔴类型1：临近退休型】【💡特征：得过且过、消极怠工】`

> 有的干部干工作投机取巧、偷奸耍滑。

`【🔴类型2：投机取巧型】【💡特征：有利抢着干、不利绕着走】`

## 八、应用场景

### 适用场景
- 国考/省考申论材料分析
- 遴选考试材料阅读理解
- 遴选作文素材提取
- 官方文件写作素材准备

### 不适用场景
- 文学作品鉴赏（使用文学分析技能）
- 新闻报道编辑（使用新闻编辑技能）
- 学术论文标注（使用学术写作技能）

□ 同一短句多类型是否依次写出？
□ 背景介绍是否标注"非采分点"？
□ 数据是否保留原文数字？
□ 术语是否使用原材料原文？
